# Jobs & Batch Processing

Batch processing and scheduled job applications using Java and Spring Framework for enterprise workload patterns, background processing, and integration with Azure storage services.

## Software Inventory

| Application     | Purpose                                                                                           |
| --------------- | ------------------------------------------------------------------------------------------------- |
| **blob-java**   | Java application for Azure Blob Storage operations and batch processing workflows                 |
| **blob-spring** | Spring Framework application demonstrating blob storage integration and background job processing |
