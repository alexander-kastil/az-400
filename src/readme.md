# Source Code

Sample applications and libraries demonstrating DevOps patterns and Azure services.

## Folder Structure

| Folder           | Description                                                                                                                          |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **ai**           | AI applications using Azure AI services, Semantic Kernel, and MCP frameworks for intelligent assistant and data processing scenarios |
| **angular**      | Angular frontend applications and CLI tools demonstrating modern web development patterns, frameworks, and deployment strategies     |
| **az-functions** | Azure Functions implementations in multiple languages (.NET, Python, TypeScript) for serverless compute and API scenarios            |
| **jobs**         | Batch processing and scheduled job applications using Java and Spring for enterprise workload patterns                               |
| **react**        | React-based applications demonstrating single-page application development and deployment patterns                                   |
| **services**     | Microservices and backend APIs (.NET-based) implementing catalog, order, configuration, and business logic services                  |
| **teams**        | Microsoft Teams integrations and SharePoint Framework (SPFx) solutions for collaborative applications                                |
