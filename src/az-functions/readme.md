# Azure Functions

Serverless compute functions implemented in multiple languages (.NET, Python, TypeScript) demonstrating function-based architecture, HTTP triggers, payment processing, and integration patterns.

## Software Inventory

| Component       | Purpose                                                                           |
| --------------- | --------------------------------------------------------------------------------- |
| **payment-net** | .NET Azure Functions for payment processing with HTTP triggers and business logic |
| **payment-py**  | Python Azure Functions implementation for payment operations and integrations     |
| **utils-ts**    | TypeScript utility functions and shared helpers for Azure Functions               |
