# Microservices

Microservices architecture implemented with .NET demonstrating distributed application patterns, domain-driven design, CQRS patterns, data persistence, and inter-service communication.

## Software Inventory

| Service             | Purpose                                                                                      |
| ------------------- | -------------------------------------------------------------------------------------------- |
| **catalog-service** | Product catalog microservice providing inventory management and product information APIs     |
| **order-service**   | Order processing microservice implementing business logic, CQRS patterns, and event handling |
| **config-service**  | Configuration service for centralized settings and application configuration management      |
| **prime-service**   | Prime membership service handling subscription and premium features                          |
| **food-app-common** | Shared libraries and common components for inter-service communication and domain models     |
