# Implementing Continuous Integration using GitHub Actions

GitHub Actions provides native CI/CD capabilities directly within GitHub repositories. These demos cover workflow fundamentals, building and deploying applications to Azure, and automating repository operations using the GitHub Script action.

## Demos

|                                                             |                                                                                |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------ |
| [Pipeline Basics](01-pipeline-basics/)                      | Understand workflow syntax, event triggers, and foundational workflow patterns |
| [CI/CD: .NET Web API](02-ci-cd/01-web-api/)                 | Automated build and deployment of .NET applications to Azure App Service       |
| [CI/CD: Angular Static Web App](02-ci-cd/02-static-webapp/) | Deploy Angular applications to Azure Static Web Apps with preview environments |
| [GitHub Script](03-github-script/)                          | Automate GitHub operations using Octokit REST API within workflows             |
