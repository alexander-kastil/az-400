# 01 Infrastructure & Configuration as Code

[What is Infrastructure as Code](https://docs.microsoft.com/en-us/devops/deliver/what-is-infrastructure-as-code)

## Demos

| Demo                          | Description                                                                                                                                                                                     |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Azure CLI**                 | Provision Azure infrastructure using bash scripts executed from Azure DevOps pipelines. Demonstrates parameterized naming, resource grouping, and reusable scripts for multi-stage deployments. |
| **Bicep**                     | Azure Resource Manager (ARM) template language for declarative infrastructure as code. Simplifies IaC with type-safe syntax and integrated validation within Azure Pipelines.                   |
| **Terraform**                 | HashiCorp's infrastructure as code tool integrated with Azure DevOps pipelines. Uses Workload Identity Federation for secure, keyless authentication with plan and apply workflows.             |
| **Azure Developer CLI (azd)** | Deploy containerized applications to Azure Container Apps using the Azure Developer CLI. Supports AZD templates for rapid cloud deployment and local development workflows.                     |
| **Azure Automation**          | Automation Accounts, Run-books, and Desired State Configuration (DSC) for managing Azure infrastructure lifecycle and compliance at scale.                                                      |
| **Ansible**                   | Configuration management and infrastructure automation using Ansible playbooks. Integrates with Azure to provision and configure cloud resources declaratively.                                 |
| **Chef**                      | Infrastructure automation and configuration management for Windows and Linux virtual machines in Azure using Chef cookbooks and recipes.                                                        |
