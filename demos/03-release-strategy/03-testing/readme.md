# Testing Applications

This module demonstrates various testing strategies for cloud applications, from unit tests validating code logic to load tests measuring performance under stress, and end-to-end tests verifying complete user workflows.

## Testing Strategies

|                                |                                                                                              |
| ------------------------------ | -------------------------------------------------------------------------------------------- |
| [01-unit-test](./01-unit-test) | Unit testing frameworks for .NET and Angular with coverage reporting and CI/CD integration   |
| [02-load-test](./02-load-test) | Automated load testing for performance validation using Apache JMeter and Azure Load Testing |
| [03-e2e](./03-e2e)             | End-to-end testing with Playwright for cross-browser user interaction validation             |
