# Deploying Applications to Kubernetes

Demonstrates deploying containerized applications to Azure Kubernetes Service (AKS) using both manual provisioning and automated Azure DevOps pipelines. These demos cover cluster setup, service deployment, and CI/CD integration for Kubernetes workloads.

## Demos

|                                   |                                                                                                            |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| [K8s Basics](k8s-basics/)         | Provision an AKS cluster and manually deploy API and UI services using kubectl and Kubernetes manifests    |
| [K8s Deployment](k8s-deployment/) | Automate container image build and Kubernetes deployment using Azure DevOps pipelines with ACR integration |
