# Package Repositories & Semantic Versioning

## Demos

- Understanding package repositories across different ecosystems (NuGet, npm, Maven, PyPI)
- Semantic versioning concepts and versioning strategies
- Managing dependencies across multiple package management systems

## Links & Resources

[Semantic Versioning](https://semver.org/) - Specification for versioning software releases using MAJOR.MINOR.PATCH scheme

[GitVersion Documentation](https://gitversion.net/) - Automate semantic versioning from Git history

[NuGet](https://www.nuget.org/) - Package repository for .NET and C# packages

[npm](https://www.npmjs.com/) - JavaScript and Node.js package registry

[Java Package Index](https://mvnrepository.com/) - Central repository for Java packages and Maven dependencies

[Python Package Index](https://pypi.org/) - Repository for Python packages and libraries
